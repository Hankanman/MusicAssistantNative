# Managing the Queue

The **Queue** page shows the current play queue for your selected player.

## Queue Display

Each queue item shows:

- **Position number** — order in the queue
- **Thumbnail** — album art
- **Track name** — bold if it's the currently playing track
- **Artist and album** — secondary info
- **Duration** — track length
- **Remove button** — :material-close: to remove from queue

The currently playing track is **highlighted** in the list.

## Queue Controls

### Header Controls

| Control | Action |
|---------|--------|
| Item count | Shows total items in queue |
| :material-shuffle: | Toggle shuffle mode |
| :material-repeat: | Cycle repeat: Off → All → One |
| **Clear Queue** | Remove all items (action button in toolbar) |

### Playing from Queue

Click any item in the queue to jump directly to that track.

### Removing Items

Click the :material-close: button on any queue item to remove it.

## Adding to Queue

Music is added to the queue when you play items from the Library:

| Action in Library | Queue Behavior |
|-------------------|---------------|
| Click a track | Plays immediately (replaces current) |
| Click an album | Queues all album tracks |
| Click a playlist | Queues all playlist tracks |

## Repeat Modes

| Mode | Icon | Behavior |
|------|------|----------|
| Off | :material-repeat-off: | Queue stops after the last track |
| All | :material-repeat: | Queue loops from the end back to the beginning |
| One | :material-repeat-once: | Current track repeats indefinitely |

## Shuffle

When shuffle is enabled, the queue plays tracks in random order. Toggle it with the shuffle button in the Queue header or on the Now Playing page.
