# Browsing Your Library

The **Library** page lets you explore your unified music collection from all connected providers.

## Navigation

Access the Library from the sidebar by clicking **Library**. The page has tabs along the top:

| Tab | Contents |
|-----|----------|
| **Artists** | All artists in your library |
| **Albums** | All albums with cover art, artist, and year |
| **Tracks** | Individual tracks with duration |
| **Playlists** | Playlists from all providers (Spotify, Tidal, etc.) |
| **Radios** | Internet radio stations |
| **Search** | Appears when you enter a search query |

## Browsing

Each tab shows a scrollable list of items. For each item you can see:

- **Thumbnail** — album art or artist photo (loaded from Music Assistant's image proxy)
- **Name** — track, album, or artist name
- **Details** — artist name, album name, or other metadata depending on the type
- **Duration** — for tracks, shown as `m:ss`
- **Favorite indicator** — a star icon for favorited items

## Playing Music

Click any item to immediately play it on your currently selected player:

- **Click a track** — plays that track
- **Click an album** — plays the album
- **Click a playlist** — plays the playlist
- **Click an artist** — plays tracks by that artist
- **Click a radio** — tunes to that radio station

## Searching

1. Type your query in the **search bar** at the top of the Library page
2. Press ++enter++
3. The **Search** tab appears with results across all media types
4. Results include artists, albums, tracks, playlists, and radios that match your query

Search queries are sent to Music Assistant, which searches across all your configured music providers simultaneously.

## Pull to Refresh

Swipe down (or use the refresh gesture) to reload the current library tab from the server.
